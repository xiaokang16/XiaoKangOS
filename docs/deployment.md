# XiaoKangOS 部署指南 - 手把手教学

## 这篇指南是给谁看的？

**完全不懂技术的小白！**

我会用最简单的大白话，手把手教你把XiaoKangOS用起来。

---

## 什么是"部署"？

"部署"听起来很专业，其实就是：
- **把系统放到电脑上运行**

就像你买了一个新手机，要"部署"它，就是开机、设置，然后开始用。

XiaoKangOS的部署有3种方式：
1. 🖥️ 在虚拟机里运行（推荐新手）
2. 💿 做成U盘启动（推荐体验）
3. 💾 安装到硬盘（推荐长期使用）

---

## 方式一：虚拟机运行（最简单的开始！）

### 什么是虚拟机？

**虚拟机就是电脑上运行的"电脑里的电脑"**。

就像你有了一台电脑，还想再要一台，但又没钱买怎么办？
→ 用软件"模拟"一台！

### 需要的工具

下载这个软件（免费）：
- **VirtualBox** - https://www.virtualbox.org/

### 具体步骤

**第一步：下载VirtualBox**
1. 打开上面的链接
2. 点击 "Download VirtualBox"
3. 下载 Windows 版本的安装包
4. 安装（一直点"下一步"就行）

**第二步：创建虚拟机**

1. 打开VirtualBox，点击"新建"

2. 填写这些：
   - 名称：`XiaoKangOS`
   - 类型：`Linux`
   - 版本：`Other Linux (64-bit)`
   - 点击"下一步"

3. 分配内存：
   - 建议分配 **2048 MB**（2GB）
   - 点击"下一步"

4. 创建硬盘：
   - 选择"现在创建虚拟硬盘"
   - 点击"创建"
   - 类型选择 VDI
   - 选择"动态分配"
   - 大小填写 **20 GB**
   - 点击"创建"

**第三步：放入ISO文件**

1. 选中左边你创建的虚拟机
2. 点击上面的"设置"
3. 选择"存储"
4. 点击"空"的光盘图标
5. 点击右边的光盘图标，选择"选择虚拟光盘文件"
6. 找到你的 `XiaoKangOS.iso` 文件
7. 点击"确定"

**第四步：启动！**

1. 选中虚拟机
2. 点击绿色的"启动"按钮
3. 等待启动......

**然后你就会看到一个界面！恭喜！🎉**

## 方式二：做成U盘启动

### 什么是U盘启动？

想象一下：
- 普通的U盘只能存文件
- 启动U盘可以"代替电脑硬盘"，开机就能用

### 需要什么？

- 一个 **4GB以上** 的U盘（数据会丢失，请先备份！）
- 下载好的 `XiaoKangOS.iso` 文件

### 具体步骤（Windows）

**方法A：用Rufus（简单）**

1. 下载Rufus：https://rufus.ie/ （很小，只有1MB）
2. 插入U盘
3. 打开Rufus
4. 在"设备"下拉框选择你的U盘
5. 点击"选择"，找到你的ISO文件
6. 其他保持默认，点击"开始"
7. 等待完成...

**方法B：用Etcher（也很简单）**

1. 下载Etcher：https://etcher.balena.io/
2. 插入U盘
3. 打开Etcher
4. 点击"Flash from file"，选择ISO
5. 点击"Select target"，选择你的U盘
6. 点击"Flash"，等待完成

**怎么从U盘启动？**

1. 关机
2. 插入U盘
3. 开机时按 **F12** 或 **Del** 或 **F2**（不同电脑不一样）
4. 选择从U盘启动
5. 搞定！

## 方式三：安装到硬盘

### 适合谁？

- 想要长期使用XiaoKangOS
- 电脑有额外空间
- 不怕格式化（会删除原有系统！）

### ⚠️ 警告

**安装会格式化硬盘！请先备份重要数据！**

### 具体步骤

**第一步：从U盘启动**

按上面的方法做成启动U盘，然后从U盘启动。

**第二步：进入安装界面**

看到界面后，按 `Ctrl + Alt + F2` 打开终端。

**第三步：格式化硬盘**

```bash
# 查看硬盘
fdisk -l

# 格式化（假设你只有一个硬盘 /dev/sda）
fdisk /dev/sda

# 在fdisk里：
# 1. 按 n 创建新分区
# 2. 按 p 创建主分区
# 3. 分区号选 1
# 4. 大小：+10G（给系统用）

# 再创建一个分区存数据：
# 1. 按 n
# 2. 按 p
# 3. 分区号选 2
# 4. 大小：+5G（给数据用）

# 按 w 保存退出
```

**第四步：挂载和安装**

```bash
# 格式化分区
mkfs.ext4 /dev/sda1
mkfs.ext4 /dev/sda2

# 挂载
mount /dev/sda1 /mnt
mkdir -p /mnt/boot
mount /dev/sda1 /mnt/boot

# 安装GRUB（启动引导器）
grub-install --root-directory=/mnt /dev/sda
```

## 首次使用指南

### 开机后的界面

启动后，你会看到：
- 一个全屏的网页
- 上面有4个选项卡：仪表盘、文件、进程、设置

### 各个功能

**仪表盘** - 显示电脑状态
- CPU使用率（绿色条）
- 内存使用（蓝色条）
- 硬盘空间
- 网络状态

**文件** - 文件管理器
- 可以浏览文件
- 双击打开文件

**进程** - 任务管理器
- 显示所有运行的程序
- 可以结束程序

**设置** - 系统设置
- 改主题颜色
- 改刷新间隔

## 常见问题

### Q: 虚拟机打不开？

**A**: 
1. 确认ISO文件路径正确
2. 重启VirtualBox
3. 给虚拟机分配更多内存

### Q: 启动很慢？

**A**: 第一次启动需要初始化，慢慢等

### Q: 没有网络？

**A**: 
1. 在VirtualBox里：设置 → 网络 → NAT
2. 真机的话检查网线/WiFi

### Q: 怎么关机？

**A**: 
- 在终端输入：`shutdown -h now`
- 或者按电源键

### Q: 中文显示乱码？

**A**: 
- 目前版本默认英文界面
- 可以在设置里调整

## 进阶使用

### 修改网页内容

想要用自己的网页代替默认界面？

```bash
# 进入系统后，按 Ctrl+Alt+F2 打开终端
# 输入用户名 root，密码 root

# 重新挂载为可写
mount -o remount,rw /

# 修改网页文件
nano /www/html/index.html
```

### 开启WiFi

```bash
# 查看网卡
ip link show

# 启用WiFi
ip link set wlan0 up

# 扫描WiFi
iwlist wlan0 scan

# 连接WiFi（需要配置wpa_supplicant）
```

### 访问终端

按 `Ctrl + Alt + F2` 切换到终端
- 用户名：`root`
- 密码：`root`

按 `Ctrl + Alt + F1` 回到图形界面
